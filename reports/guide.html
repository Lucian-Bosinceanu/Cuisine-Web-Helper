1. Abstract

We are a team of students from "Faculty of Computer Science" Iași, IInd year, B1 group, and this is our Web Technologies project - Cuisine Web Helper. This article provides instructions on how to use this application, such as core features and admin & user functionalities.

2. Introduction

Cuisine Web Helper is a Web application that allows people to quickly find cooking recipes for their next delicious meal. Each recipe has multiple tags associated with it in regard to the ingredients used, cooking methods and tools, food restrictions, life style etc. The search engine is based on providing recipes that satisfies a list of tags provided by the user. In addition to this, the users have access to a series of articles that they can read in order to hone their cooking skills or learn something new.

3. Main functionalities

Cuisine Web Helper has two types of users: normal users and administrators. Both of them have a set of common functionalities such as login and search for resources (recipes or articles), as described below. In addition, admins are responsible with content management. They can create, update and delete resources. A normal user cannot modify application's content, but it has access to a profile page where special food restrictions and lifestyles can be checked in order to get appropriate recipes. A common usage flow for a normal user is as its follows: open application page, go to register and create account, go to profile and set preferences, return to main page and search for recipes.

4.1. Register

Anyone can register to the application and become normal users. From the main page, locate the menu in the upper right. Press the Register button and go to register page. Instert your desired username, email and password, then click "Register". You will be redirected to home page. Please note that admin accounts cannot be created through this method. To add a new admin to the system, upgrade a normal user account to admin status directly from database by switching the is_admin flag from 0 to 1.
